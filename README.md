# NewDiary

## どんなアプリ？

このアプリはシンプルな日記です。そして Swift を勉強し始めてから初の自作アプリです。

## 機能

- 日記の一覧表示機能
  - カードをドラッグすることによって他の日記を見ることができます。
  - 左上のリターンボタンを押すことでドラッグして消えたカードを戻すことができます。
- 日記の作成機能
  - タイトルと文章を入力します。
  - 入力時の日付が自動的に反映されます。
